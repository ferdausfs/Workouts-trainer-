# Fonts

This app uses **Poppins** (Open Font License) loaded primarily via the `google_fonts` package at runtime.

For offline-bundled fonts, download from https://fonts.google.com/specimen/Poppins and place:

- `Poppins-Regular.ttf`
- `Poppins-Medium.ttf`
- `Poppins-SemiBold.ttf`
- `Poppins-Bold.ttf`

into this folder. They are referenced from `pubspec.yaml`.
